📦 AWS S3 Setup for Wallpapers:
1. Create an S3 bucket: hussain-portfolio-assets
2. Enable static website hosting + CloudFront CDN
3. Upload wallpapers as: wallpapers/wallpaper-01.jpg (recommended size: 1290×2796 for iPhone 14 Pro)
4. Update each href and data-src in wallpapers.html with your S3/CloudFront URLs
5. Set bucket CORS policy to allow downloads from your domain
